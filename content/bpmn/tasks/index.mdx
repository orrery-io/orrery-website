# Tasks

Tasks represent units of work in a BPMN process. Select a task type to learn more.

| Task | Description |
|---|---|
| [Service Task](./tasks/service-task) | Delegates work to an external worker via topic subscription |
| [Script Task](./tasks/script-task) | Executes an inline expression to compute a variable |
| [Receive Task](./tasks/receive-task) | Waits for a named message to be delivered |
| [Multi-Instance](./tasks/multi-instance) | Runs a task once per item in a collection |
