# BPMN Elements

Orrery supports the BPMN 2.0 standard. This section documents each supported element, its behavior, and configuration.

## Element categories

- **[Tasks](/bpmn/tasks)** — Units of work (service, script, multi-instance)
- **[Gateways](/bpmn/gateways)** — Flow control (exclusive, parallel, inclusive, event-based)
- **[Events](/bpmn/events)** — Start, end, intermediate, and boundary events
- **[Subprocesses](/bpmn/subprocesses)** — Nested processes and event subprocesses
