# Gateways

Gateways control the flow of tokens — splitting or merging paths based on conditions or events.

| Gateway | Description |
|---|---|
| [Exclusive (XOR)](./gateways/exclusive) | Routes to exactly one outgoing path based on conditions |
| [Parallel (AND)](./gateways/parallel) | Forks all paths simultaneously; joins when all arrive |
| [Inclusive (OR)](./gateways/inclusive) | Activates all paths whose condition is true |
| [Event-Based](./gateways/event-based) | Waits for the first of several competing events |
