# Subprocesses

Subprocesses embed nested flows and handle scope, errors, and event-driven interruptions.

| Element | Description |
|---|---|
| [Subprocess](./subprocesses/subprocess) | Embeds a complete BPMN flow inside a parent process |
| [Event Subprocess](./subprocesses/event-subprocess) | Triggered by an event while the parent scope is active |
| [Link Events](./subprocesses/link-events) | GOTO jump between two points within the same process level |
