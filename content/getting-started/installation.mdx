# Installation

## Docker (recommended)

Orrery is published to the GitHub Container Registry. Pull the latest image:

```bash
docker pull ghcr.io/orrery-io/orrery:latest
```

Run with a PostgreSQL connection string:

```bash
docker run -e DATABASE_URL=postgres://... -p 3000:3000 ghcr.io/orrery-io/orrery
```

All published versions are listed at [ghcr.io/orrery-io/orrery](https://github.com/orrery-io/orrery/pkgs/container/orrery).

## From source

```bash
git clone https://github.com/orrery-io/orrery
cd orrery
cargo build --release
./target/release/orrery-server
```

## Configuration

| Environment Variable | Default | Description |
|---|---|---|
| `DATABASE_URL` | required | PostgreSQL connection string |
| `PORT` | `3000` | HTTP server port |
