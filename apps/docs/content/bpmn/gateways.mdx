# Gateways

Gateways control the flow of execution through a process.

## Exclusive Gateway (XOR)

Routes to exactly one outgoing path based on conditions.

```xml
<exclusiveGateway id="checkAmount" name="Amount OK?" />
<sequenceFlow sourceRef="checkAmount" targetRef="approve">
  <conditionExpression>${amount > 0}</conditionExpression>
</sequenceFlow>
```

## Parallel Gateway (AND)

Splits into multiple parallel paths (fork) or waits for all paths to complete (join).

```xml
<parallelGateway id="fork" />
<parallelGateway id="join" />
```

## Inclusive Gateway (OR)

Routes to one or more outgoing paths. All matching conditions execute.

```xml
<inclusiveGateway id="routeOrder" />
```

## Event-Based Gateway

Waits for the first of several events (messages, timers, signals) and routes accordingly.

```xml
<eventBasedGateway id="waitForResponse" />
```
