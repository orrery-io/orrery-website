# Events

Events represent things that happen during a process — starts, waits, and endings.

| Event | Description |
|---|---|
| [Start & End Events](./events/start-end) | Begin and terminate process instances |
| [Timers](./events/timers) | Pause execution for a duration, until a date, or on a schedule |
| [Messages](./events/messages) | Pause execution until a correlated message is delivered |
| [Signals](./events/signals) | Broadcast to all matching subscriptions across instances |
| [Boundary Events](./events/boundary) | Attach to tasks to handle errors, timeouts, and messages |
